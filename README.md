# UrBackup Monitoring

Advanced monitoring and analytics layer for UrBackup.

## Features

- Backup health monitoring
- Live job tracking
- Log analysis (error detection)
- Storage usage tracking
- Incremental backup analytics

{
  "client": "PC-523",
